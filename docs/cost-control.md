# Cost Control

This file documents cost control actions for the Azure Cloud Operations Lab.

## Student Credit Awareness

This project was created using Azure student credit.

To avoid unnecessary charges, resources should not remain running longer than needed.

## Cost Control Actions

- Used a small lab environment.
- Avoided unnecessary services.
- Took screenshots after each completed step.
- Planned to delete the resource group after finishing the lab.
- Avoided leaving the VM running overnight.

## Important Resources to Delete

When finished, delete the resource group:

`rg-cloud-ops-lab`

This removes:

- Virtual machine
- Disk
- Public IP address
- Virtual network
- Network security group

## Result

The project demonstrates cost awareness, which is important for real cloud operations work.
